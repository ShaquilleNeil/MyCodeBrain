This project parses the Canadian Tech Internships dataset from a public GitHub README and converts it into structured, usable data.

## What it does

- Fetches raw markdown from a GitHub README  
- Extracts internship listings from tables  
- Cleans and normalizes the data  
- Handles nested fields (links inside text)  
- Supports search and pagination  

## Why I built this

This was part of my weekly CodeBrain challenges, where I focus on building small projects to improve real-world problem solving.

The goal wasn’t just to display data, but to make unstructured data usable for applications.
